Python Flashcard Project.
Webbased Flashcard Learning App written with Flask.

Features
Important features include register, login, addition of Flashcard Deck,Learn the collection,etc
All the html Templates are present in templates folder and main.py contains the code for Flask App which renders the whole WebApp. There is a python file to create a Database for the Application.

Installation

1.Install the Requirements - pip3 install -r requriements/requirements.txt

2.Setup the Database - python create-db-table.py

3.Run the Application - python main.py

the Application was created and tested with Python Version 3.8
